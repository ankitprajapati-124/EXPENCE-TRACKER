
# EXPENCE-TRACKER 

A **simple Expense Tracker Application** to help users record and manage their everyday expenses using HTML, CSS & JavaScript.

# Description

EXPENCE-TRACKER is a web-based tool that lets users enter, view, and manage their daily expenses. Users can track the amount spent, add expense details, and monitor their overall spending to better manage personal finances.

It is built with **HTML**, **CSS**, and **JavaScript**, and provides a clean interface for tracking and organizing expenses with ease.


# Features

✔ Add new expenses  
✔ View all added expenses  
✔ Delete expense entries  
✔ Displays total expense amount  
✔ Data persists (if implemented with local storage)  
✔ Simple and responsive design
